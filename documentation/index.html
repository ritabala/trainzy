Gym Management

Manage all fitness centers in the system with comprehensive gym management features.

Add New Gym

Create new fitness centers with complete setup:

  • • Basic gym information (name, address, contact details)
  • • Logo and branding customization
  • • Timezone and currency settings
  • • Admin user creation with automatic role assignment
  • • Package assignment and subscription setup

Edit Gym Details

Modify existing gym information and settings:

  • • Update gym profile and contact information
  • • Change logo and branding elements
  • • Modify timezone and currency preferences
  • • Update package subscriptions

Gym Impersonation

Access gym admin panels directly:

  • • Seamlessly switch to gym admin view
  • • Troubleshoot gym-specific issues
  • • Provide direct support to gym administrators
  • • Monitor gym operations in real-time

Gym Deletion

Remove gyms from the system when needed:

  • • Permanent removal of gym data
  • • Automatic cleanup of associated records
  • • Confirmation prompts for safety

Package Management

Create and manage subscription packages that gyms can purchase to access different features and modules.

Package Types

Free Plans
  • • Default Package (basic features)
  • • Trial Package (time-limited access)
  • • Limited member and staff capacity
Paid Plans
  • • Standard Package (monthly/annual billing)
  • • Lifetime Package (one-time payment)
  • • Full feature access

Package Configuration

  • • Set pricing for different billing cycles
  • • Configure member, staff, and class limits
  • • Select available modules and features
  • • Define trial periods and notifications
  • • Integrate with payment gateways (Stripe)

Module Management

Control which features are available in each package:

  • • Core modules (members, staff, classes)
  • • Additional modules (reports, advanced features)
  • • Module activation/deactivation
  • • Feature-based access control

System Settings

Configure global system settings that affect all fitness centers in the platform.

Application Settings

  • • Global application name and branding
  • • System logo and favicon
  • • Default currency configuration
  • • Language and localization settings

Currency Management

  • • Add and manage supported currencies
  • • Set currency symbols and codes
  • • Configure exchange rates
  • • Enable/disable currencies for gyms

Payment Gateway Configuration

  • • Stripe integration setup
  • • API key configuration
  • • Webhook endpoint management
  • • Payment gateway status monitoring

Payment Management

Monitor and manage all payment activities across the platform.

Gym Subscription History

  • • View all gym package subscriptions
  • • Track subscription status and changes
  • • Monitor billing cycles and renewals
  • • Access subscription analytics

Payment Tracking

  • • View all gym payments across the platform
  • • Monitor payment status and history
  • • Track revenue and financial metrics
  • • Generate payment reports

Offline Payment Requests

  • • Manage offline payment requests from gyms
  • • Approve or reject payment confirmations
  • • Track manual payment processing
  • • Maintain payment audit trail
